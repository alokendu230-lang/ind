HTML (Hyper Text Markup Language)


    HTML stands for HyperText Markup Language. It is the standard language used to create and structure webpages. HTML provides the basic building blocks for websites by defining elements like headings, paragraphs, images, links, tables, and forms.

  • HyperText: Refers to the ability to link documents together using hyperlinks, allowing users to navigate between pages easily.

  • Markup: Means using tags to define elements within a document. These tags tell the browser how to display the content.

  • Language: Indicates that it is a set of rules and syntax used to write and interpret web content.

    • HTML works closely with CSS (for styling and design) and JavaScript (for interactivity) to create complete, functional, and visually appealing websites. It forms the foundation of front-end development.

  1. THE HEADING

    2. Headings are HTML tags used to define titles and subtitles on a webpage. They help organize content and create a logical hierarchy, making it easier for users to read and for search engines to understand the structure of the page. There are six levels of headings, ranging from h1 to h6:
    • h1 represents the main title or the most important heading on the page
    • h2 to h6 are used for subheadings, with importance decreasing as the number increases.

    • Main Title of the Page-----h1 tag

    • Section Heading---h2 tag

    • Subsection Heading---h3 tag

    • Smaller Subsection Heading----h4 tag

    • Even Smaller Subsection Heading----h5 tag
    • Least Important Heading----h6 tag

  2. THE PARAGRAPH

    3. The paragraph tag is one of the most commonly used HTML tags. It is used for grouping sentences or blocks of text together into a distinct paragraph.

    The paragraph tag defines a separate paragraph of text. Browsers automatically add space before and after a paragraph to separate it from other content, making the text easier to read.   3. HTML HORIZANTAL RULES

    4. The horizontal rule tag in HTML is used to create a horizontal line across the page. It is often used to separate sections of content visually.

    Purpose

    • To divide content into distinct sections.
    • To improve readability by adding a visual break between topics.
    • It is a self-closing tag, meaning it does not need a closing tag.


  4. THE LINE BREAK TAG

    5. The line break tag in HTML is used to insert a single line break within text. It is helpful when you want to start a new line without creating a new paragraph

    Purpose:

    • To break the text onto the next line.
    • Useful for addresses, poems, or any content that requires line spacing without a full paragraph
    • It is a self-closing tag, so it does not need a closing tag


  5. LIST ELEMENTS
    6. These tags are used to create lists. This is a perfect example of nesting.

    The Tags:

    • ul:An Unordered list It creates a bulleted list
    • ol: An Ordered List. It creates a numbered list.
    • li: A list Item. Each item in either type of list must be wrapped in an litag.

    Purpose:To group related items together in a list format


  6. THE ANCHOR TAG

    7. The anchor tag, is used to create hyperlinks in HTML. Hyperlinks allow users to navigate from one webpage to another, open a file, or jump to a specific location within the same page. This is what makes the web "hypertext," connecting pages and resources across the internet.

    Purpose:

    The main purpose of the anchor tag is to make text or images clickable, so users can go to a different page or section. To work, it requires an href attribute (hypertext reference) that specifies the URL or location of the destination.

    HTML Links - The target Attribute

    By default, the linked page will be displayed in the current browser window. To change this, you must specify another target for the link.

    The target attribute specifies where to open the linked document

    The targetattribute can have one of the following values:

    alues:
    • self-Default. Opens the document in the same window/tab as it was clicked
    • blank - Opens the document in a new window or tab

  7. THE IMAGE TAG

The image is a self-closing tag used to embed an image onto a webpage.

The Purpose:

The imgage s used to display a visual image on your page. It requires two main attributes:

  • src (source): Specifies the path or URL to the image file. This is mandatory
  • alt (alternative text): Provides a description of the image. This is important for accessibility (screen readers for visually impaired users will read this) and is also displayed if the image fails to load
