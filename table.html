8. HTML NESTED LISTS AND TABLES

  • Nested Lists in HTML

    • A nested list is a type of list in HTML where one list is placed inside another list, creating a hierarchy of items. It helps organize information by showing main items along with their related sub-items. For example, you might have a main category like "Fruits" and, inside it, a list of specific fruits such as "Apples," "Bananas," and "Oranges." Nested lists can be made using either an unordered list (ul) with bullet points, an ordered list (ol) with numbers, or a combination of both. The inner list must always be placed inside a (li) (list item) tag of the outer list. Nested lists are useful for menus, outlines, recipes, or any content where a clear structure and hierarchy are needed.

    Types of Nested Lists

    • Nested lists are used to organize information in a structured way by placing one list inside another. There are three main types of nested lists in HTML, based on the combination of lists used:


    1. Nested Unordered List (UL inside UL)

    In this type, both the outer and inner lists use bullet points. It is commonly used for categories and subcategories.

    Example:

    SHOPPING LISTS
    • Fruits
      • Apple
      • Banana
      • Orange
    • Vegetables
      • Carrot
      • Potato

    2.Mixed Nested List (UL inside OL or OL inside UL)

    In this type, one list uses numbers and the other uses bullets. This is often used when main steps are numbered, and sub-items are listed with bullet points.

    Country List

    • India
      • Maharashtra
        1. Mumbai
        2. Pune
      • Karnataka
      • Tamil Nadu
    • Japan
      • Tokyo
      • Hokkaido
        1. Sapporo
        2. Asahikawa
        3. Hakodate

    3. Nested Ordered List (OL inside OL)

    Here, both the outer and inner lists are numbered. It is useful when there is a strict step-by-step hierarchy.
    Example:

    1. Chapter 1(PROFIT AND LOSS)
      1. Introduction
      2. Basics
    2. Chapter 2(PERENTAGE)
      1. Advanced Topics
      2. Summary
